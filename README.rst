Usage
-----
1. Sign in to https://gumiya.pmrowla.com/
2. Link and verify your osu! account
3. Configure and enable the bot


Features
--------
* Beatmap requests
* PP info for requests via Tillerino
* Optional subscribers only mode for map requests
